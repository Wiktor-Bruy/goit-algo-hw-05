# goit-algo-hw-05

This is a small project I worked on during my studies.

## Project capabilities

- **task_1** - contains the **_caching_fibonacci_** function, which returns another **_fibonacci_** function as its result. It takes an integer as an argument and returns the Fibonacci number corresponding to the specified ordinal number.
- **task_2** - contains the **_sum_profit_** function. It accepts text and a function as a second argument, which is also defined in the module. It scans the test for float numbers. The function returns the sum of these numbers. The numbers must be separated by spaces on both sides.
- **task_3** - a script that expects the path to a log file as its first argument and the optional logging level as its second. It outputs logs from the file, specifying their number. If you specify a second argument, it will also output logs of the specified level.
- **main** - a script for a console application that helps maintain a contact list.
